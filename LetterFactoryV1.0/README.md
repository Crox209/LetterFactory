LetterFactory 1.0

Overview
- Load a .docx template containing placeholders like [[Name]] or [[Date]].
- LetterFactory will detect placeholders and create input fields automatically.
- Export a finished document as .docx or .pdf, or print it directly.

Single Document Mode
1) Click Template and select a .docx template.
2) Fill in all detected placeholder fields.
3) Choose output format (.docx or .pdf).
4) Select an output folder (optional; defaults to the app folder).
5) Click Export or Print.

Tip: You can use tab to quickly switch to the next blank or easily replace whats inside

Bulk Mode
1) Load a template first.
2) Click Bulk Mode and upload an Excel .xlsx file.
3) Ensure Row 1 headers match placeholders exactly (including brackets), e.g. [[Name]].
4) Select placeholders to build each output filename (optional).
5) Optionally enable “Merge all into one file”.
6) Click Generate All.

Notes
- This app does not save any settings when it closes.
- Placeholders are case-sensitive: [[Name]] and [[name]] are different.
